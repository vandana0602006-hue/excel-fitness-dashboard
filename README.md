# Excel Fitness Dashboard

## 📊 Project Overview
A data-driven Excel fitness tracker designed to monitor daily activity, weight, and workout progress using interactive dashboards and charts.

## 🔧 Features
- Daily fitness tracking
- Weight monitoring
- Goal tracking
- Data visualization using charts
- Interactive dashboard

## 🧠 Skills Used
- Microsoft Excel
- Data Analysis
- Pivot Tables
- Charts & Graphs
- Dashboard Creation

## 🎯 Purpose
This project helps in analyzing fitness progress and making better health decisions through data visualization.
